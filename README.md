# FriskStudio Code – Unity Edition  
Unity 専用のブラウザ IDE。

このエディションは **Unity 開発に必要な主要ファイルすべて** を扱える  
フル機能の IDE です。

対応言語は以下の 5 系統：

- Unity C#  
- Unity Shader（ShaderLab / HLSL）  
- Unity YAML（Scene / Prefab / Animator）  
- manifest.json（Unity Package Manager）  
- package.json（Unity Package）  

Unity Editor 公式風の UI を採用し、  
C# テンプレート、Shader プレビュー、Unity Console など  
Unity 開発に特化した機能を搭載しています。

---

## 🎮 特徴

### ✔ Unity C# テンプレート内蔵
以下のテンプレートをワンクリックで切り替え可能：

- MonoBehaviour  
- ScriptableObject  
- Editor 拡張  
- Custom Inspector  
- Runtime Script  
- Event System  
- UI Script（Button / TMP）  

### ✔ Unity Shader（ShaderLab / HLSL）対応
- Surface Shader  
- URP Shader  
- HDRP Shader  
- Compute Shader  
- Shader Preview（簡易解析）  

### ✔ Unity YAML 対応
- Scene YAML  
- Prefab YAML  
- Animator Controller YAML  

### ✔ Unity Package / manifest.json 対応
- package.json  
- manifest.json  

### ✔ Unity Console（Unity 風ログビュー）
- Debug.Log  
- Debug.LogWarning  
- Debug.LogError  
色分けされた Unity Editor 風のログ表示。

### ✔ Monaco Editor ベース
- シンタックスハイライト  
- 自動レイアウト  
- ダーク / ライトテーマ切替  
- ファイルダウンロード機能（DL ボタン）

---

## 🧩 使い方

1. 起動画面で編集したい Unity ファイル種別を選択  
   - Unity C#  
   - Unity Shader  
   - Scene YAML  
   - Prefab YAML  
   - Animator YAML  
   - manifest.json  
   - package.json  
2. エディタ画面が開く  
3. Unity C# の場合はテンプレートをプルダウンで変更可能  
4. Shader の場合は右側に Shader Preview が表示  
5. Unity Console にログを出力可能  
6. 「DL」ボタンでファイルを保存
